Introduction

This module is designed to simplify the process of recording minutes or notes. E-Notes is a system that converts audiovisual speech into text, eliminating the need for manual input.

Recording

The first step in using the E-Notes addon is to press the Start Recording button. Your spoken words will automatically be transcribed into text in the designated text field. The Pause button pauses the recording, while the Title field allows you to enter a title. The Save button saves the recorded data into the Odoo database.

Tree View

Recorded and saved data will be stored and displayed in the tree view under the Notes menu.

Form View

The form view includes several fields to be filled out as additional data beyond meeting notes, such as date, title, location, participants, and an option to upload documents. There are also fields for recording conclusions and follow-up actions. The Note text field, which contains the transcription of recorded speech, can be edited and adjusted as needed.
