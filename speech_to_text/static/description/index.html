Introduction

Module ini gunanya untuk memudahkan dalam penginputan enotulen atau notes, E-Notes yaitu suatu sistem yang mengubah ucapan audiovisual menjadi text sehingga tidak perlu input manual.

Recording

Langkah pertama dalam menggunakan addons E-Notes adalah dengan cara menekan tombol Start Recording, suara atau ucapan yang kita keluarkan akan otomatis masuk menjadi text di kolom text tersebut. Tombol Pause untuk Pause, dan kolom Title untuk memasukan judul. Tombol Save berfungsi untuk menyimpan data hasil record ke dalam database Odoo.

Tree View

Data yang telah terecord dan di save akan masuk tersimpan di tampilan tree tersebut pada menu Notes.

Form View

Di tampilan form terdapat beberapa kolom untuk diisi sebagai data-data tambahan selain data catatan hasil rapat, seperti data tanggal, title, lokasi, peserta dan fitur untuk upload dokumen, ada juga kolom untuk mencatat kesimpulan dan tindak lanjut. Di kolom Note text hasil record dari suara atau ucapan bisa di edit dan disesuaikan.
